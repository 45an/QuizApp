## üëΩ Features

   -  Quiz Creation: Users can create their own quizzes through a simple interface.
   -  Quiz Participation: Users can participate in existing quizzes by clicking on a link or searching for quizzes.
   -  Quiz Sharing: Users can share their created quizzes by generating unique links that can be shared via social media or email.
   -  User-Friendly Interface: The application features an intuitive and simple user interface for seamless navigation.

   -  User Identification: Login or user identification can be offered to save quizzes, view previous results, and personalize the experience.
   -  Responsive Design: The application is designed to work smoothly on various devices and screen sizes for a consistent experience across platforms.

## üñ•Ô∏è API Documentation

### These API endpoints handle the addition and retrieval of quizzes. 
-  The "addquiz" endpoint receives a POST request with a Quiz object in the request body. It then creates a new Quiz entity in the database based on the provided data. If successful, it returns the added quiz in the response. If an error occurs, it returns a 500 Internal Server Error.

- The "getallquizzes" endpoint retrieves all quizzes from the database, including associated media. It then converts each quiz into a QuizView object and returns them in the response.

### Here's a short & clear description:

- **addquiz Endpoint**: Adds a new quiz to the database based on the provided Quiz object in the request body. Returns the added quiz if successful, otherwise returns a 500 Internal Server Error.
  
- **getallquizzes Endpoint**: Retrieves all quizzes from the database, including associated media, and returns them as a list of QuizView objects in the response.

These endpoints facilitate the management of quizzes, including creation and retrieval.

 [HttpGet("getallquizzes")]
 public IActionResult GetAllQuizzes()
 {
     var quizzes = _context.Quizzes?.Include(q => q.Media).ToList();

     List<QuizView> quizzesView = new List<QuizView>();

     foreach (var quiz in quizzes)
     {
         quizzesView.Add(QuizConverter.Convert(quiz));
     }

     return Ok(quizzesView);
 }


  [HttpPost("addquiz")]
 public async Task<ActionResult> AddQuiz([FromBody] Quiz quizModel)
 {
     var userId = User.FindFirst(ClaimTypes.NameIdentifier).Value;
     var user = await _userManager.FindByIdAsync(userId);

     try
     {
         // Create a new Quiz object and add it to the database
         var quizToAdd = new Quiz
         {
             UserId = userId,
             UserName = user.UserName,
             Title = quizModel.Title,
             DateCreated = DateTime.UtcNow,
             MaxScore = quizModel.Questions.Count * 100, // Assuming each question is worth 100 points
             GamesPlayed = 0,
             Questions = new List<Question>() // Initialize the Questions collection
         };

         if (quizModel.Media != null && quizModel.Media.Hash != null)
         {
             // Fetch media based on the provided Hash
             var media = _context.Media?.FirstOrDefault(m => m.Hash == quizModel.Media.Hash);
             // Associate media with the quiz
             quizToAdd.Media = media;
         }

         foreach (var questionModel in quizModel.Questions)
         {
             var questionToAdd = new Question
             {
                 Questions = questionModel.Questions,
                 Answer = questionModel.Answer,
                 MultipleChoice = questionModel.MultipleChoice,
                 QuizId = quizToAdd.Id, // Set QuizId for the question
                 MocksAnswers = new List<Mock>()
             };

             if (questionModel.Media != null && questionModel.Media.Hash != null)
             {
                 // Fetch media based on the provided Hash
                 var mediaQuestion = _context.Media?.FirstOrDefault(m =>
                     m.Hash == questionModel.Media.Hash
                 );
                 // Associate media with the question
                 questionToAdd.Media = mediaQuestion;
             }

             // Add mock answers if it's a multiple choice question
             if (questionModel.MultipleChoice)
             {
                 foreach (var mockAnswerModel in questionModel.MocksAnswers)
                 {
                     var mockToAdd = new Mock
                     {
                         QuestionId = questionModel.Id,
                         MockAnswer = mockAnswerModel.MockAnswer
                     };
                     questionToAdd.MocksAnswers.Add(mockToAdd);
                 }
             }

             quizToAdd.Questions.Add(questionToAdd);
         }

         _context.Quizzes?.Add(quizToAdd);
         await _context.SaveChangesAsync();

         var quizView = QuizConverter.Convert(quizModel);

         return Ok(quizView);
     }
     catch (Exception ex)
     {
         // Log the exception for debugging purposes
         Console.WriteLine($"An error occurred while adding quiz: {ex.Message}");
         return StatusCode(
             500,
             "An error occurred while adding quiz. Please try again later."
         );
     }
 }
